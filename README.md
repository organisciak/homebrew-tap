# Homebrew Tap for atmux

Homebrew formulae for [atmux](https://github.com/organisciak/atmux) - per-project tmux sessions tuned for AI coding.

## Usage

```bash
brew tap organisciak/tap
brew install atmux
```

## Available Formulae

- **atmux** - Per-project tmux sessions for AI coding agents

## About

atmux (agent-tmux) gives you per-project tmux sessions tuned for AI coding. Run it in a repo, get a session with agent panes, and hop back in later.

For more information, see the [main repository](https://github.com/organisciak/atmux).
